# 60秒招商短视频剪辑脚本

**目标平台**：抖音 / 微信视频号（9:16 竖屏）
**受众**：潜在合伙人 / 投资者
**总时长**：60 秒

---

## 脚本结构

### 1. 开场吸引：痛点切入 (0:00-0:08)
- **画面描述**：使用“店面环境（社区小店）”的全景或中景镜头，画面干净整洁，随后快速切换至“纳客小程序界面”的自动开灯/开门动画。
- **替代方案**：若无实拍，使用文字卡“你有没想过开一家不用雇人的店？”配以科技感背景。
- **字幕/文字叠层**：
  - 【你想过开一家不用雇人的店吗？】
  - 【2026 创业新风口】
- **音效/背景音乐**：节奏感强的轻快电子音乐起，伴随门锁开启的“咔哒”声。
- **旁白**：你有没有想过，开一家不用雇人的店？在厦门，现在就有这样一个机会。

### 2. 设备展示：高尔夫 + 射击 (0:08-0:25)
- **画面描述**：
  - (0:08-0:16) 快速剪辑：高尔夫挥杆慢动作 + 球飞行轨迹追踪画面 + CG3D 真实球场特写。
  - (0:16-0:25) 快速切换：射击激光定位画面 + 射击游戏画面。突出“一机两用”的切换感。
- **替代方案**：使用设备特写照片（摄像机、投影仪、打击幕）配以平滑缩放效果，叠加功能图标。
- **字幕/文字叠层**：
  - 【厦门首家：高尔夫 + 射击】
  - 【DTG3 精度分析 | 100+ 真实球场】
  - 【ShootSim 激光定位 | 沉浸式射击】
- **音效/背景音乐**：高尔夫清脆的击球声（0:10），射击游戏的音效（0:18）。音乐节奏加快。
- **旁白**：厦门首家高尔夫加射击复合体验店。顶级模拟器设备，专业级精度，一套系统两种玩法，覆盖全客群。

### 3. 商业亮点：核心数字 (0:25-0:42)
- **画面描述**：数字动画视觉化展示。背景使用“老板助手 APP”的营收数据界面截图（模糊处理）或简约的商业图标。
- **替代方案**：全屏色块文字卡，通过动态位移展示核心数字。
- **字幕/文字叠层**：
  - 【极低门槛：日均 2.4 小时即可损益平衡】
  - 【超快回本：预期 8-13 个月】
  - 【人力零负担：巡店/客服/清洁全包】
- **音效/背景音乐**：节奏变得稳重，伴随金币落下的清脆音效（0:35）。
- **旁白**：不仅好玩，更能赚钱。日均使用仅需 2.4 小时即可实现损益平衡，预期 8 到 13 个月回本。

### 4. 合作模式：轻资产经营 (0:42-0:52)
- **画面描述**：设备特写（投影仪/打击垫）叠加价格标签动画。画面展示巡店员或清洁（如有素材）的快速蒙太奇。
- **替代方案**：图标排列展示：[设备月租] [人力全包] [区域保护]。
- **字幕/文字叠层**：
  - 【设备不用买！月租 ¥1,000-1,500】
  - 【半无人经营 | 区域保护】
- **音效/背景音乐**：背景音乐推向高潮，节奏紧凑。
- **旁白**：设备不用买，月租只需 1000 到 1500 元。巡店、客服、清洁，总部全包，你只管收钱。

### 5. CTA 行动呼吁 (0:52-0:60)
- **画面描述**：定格画面。左侧为品牌 Logo，右侧为二维码占位符，下方显示联系方式。
- **替代方案**：简洁的品牌背景图。
- **字幕/文字叠层**：
  - 【扫码了解合作详情】
  - 【联系人：Annie Chen】
  - 【微信：dinchen-annie】
- **音效/背景音乐**：音乐在强拍结束，留下余韵。
- **旁白**：厦门独家合作，名额有限。立即扫描屏幕二维码，或添加微信 dinchen-annie，开启你的轻资产创业路。

## 剪辑建议
1. **节奏**：卡点剪辑，配合 BGM 鼓点进行转场。
2. **色调**：保持明亮、干净、科技感。
3. **安全区域**：文字叠层避开抖音/视频号侧边点赞评论区。
4. **禁止内容**：脚本严禁使用非本项目素材。
